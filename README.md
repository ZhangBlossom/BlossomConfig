# BlossomConfig
BlossomConfig配置中心

当前配置中心支持：

1：项目启动时动态加载解析boostrap配置文件

2：连接自研配置中心，并从配置中心拉取配置到本地进行加载

3：支持基于长轮询的方式对配置变更事件进行监听，当配置变更时，同步修改本地配置

4：基于文件与缓存的方式优化项目性能

当前项目可以作为简历项目
